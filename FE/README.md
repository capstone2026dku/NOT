# danbab_app

A new Flutter project.

## 설치 방법
아래와 같은 방법으로 설치할 수 있습니다.
1. Flutter 설치
```
cd /c
git clone https://github.com/flutter/flutter.git -b stable
```

2. 환경변수 등록
```
export PATH="$PATH:/c/flutter/bin"
source ~/.bashrc
```

3. 확인
```
which flutter
flutter doctor
```

# 실행 방법 (Google Chrome 기준)
아래와 같은 방법으로 실행할 수 있습니다.
```
cd ~/Desktop/NOT/FE
flutter pub get
flutter run -d chrome
```
